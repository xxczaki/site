# xxczaki.com

Personal website built with [Astro](https://astro.build/), styled with TailwindCSS, and deployed using Cloudflare.

## Fonts

This project uses licensed fonts (ABC Whyte and ABC Favorit Mono) which are **not included** in the repository. To build the site locally, you need to place the following files in `src/fonts/`:

- `ABCWhyteVariableEdu.woff2`
- `ABCFavoritMonoVariableEdu.woff2`
